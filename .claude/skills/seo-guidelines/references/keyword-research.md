# 关键词研究指南

## 关键词密度控制

### 原则

- 关键词自然出现，避免堆砌
- 根据页面类型控制合适的内容长度
- 保持内容可读性和用户体验

### 推荐密度

- **核心关键词**: 1-2% 的内容密度
- **相关关键词**: 自然分布在内容中
- **避免**: 同一关键词连续出现

## 关键词选择策略

### 模型页面关键词

**格式**: `{Model Name} API`

示例:
- `Nano Banana Pro API`
- `Sora 2 API`
- `GPT-4o Image API`

### 功能页面关键词

**格式**: `{Feature} API`

示例:
- `AI Image Generation API`
- `Text to Video API`
- `Image Editing API`

### 资源页面关键词

**格式**: `{Resource Type}`

示例:
- `API Pricing`
- `API Documentation`
- `API Tutorial`

## 竞品分析方法

### Step 1: 搜索目标关键词

在搜索引擎中搜索目标关键词，记录前 10 个结果。

### Step 2: 分析竞品页面

对每个竞品页面评估：
- **链接权重**: 内链数量、外链质量
- **内容信息量**: 内容深度、实用性
- **用户体验**: 页面加载速度、布局、可读性

### Step 3: 找到突破口

识别哪些竞品是你当前竞争力可以超越的：
- 内容质量不足的页面
- 用户体验差的页面
- 信息过时的页面

### Step 4: 制定超越策略

- **内容**: 提供更深入、更实用的信息
- **体验**: 优化页面加载速度和布局
- **及时性**: 保持内容更新

## 长尾关键词策略

### 什么是长尾关键词

- 更具体、更长的关键词短语
- 搜索量较低，但转化率高
- 竞争度低，更容易排名

### 示例

**主关键词**: `AI Image API`

**长尾关键词**:
- `AI image generation API with text rendering`
- `Free AI image API for developers`
- `Best AI image API for 4K output`

### 如何使用

- 在内容中自然融入长尾关键词
- 创建专门针对长尾关键词的内页
- 用长尾关键词做内链锚文本
