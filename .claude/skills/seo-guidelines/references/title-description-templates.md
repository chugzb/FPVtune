# Title 和 Description 规范

## Title 规范

### 格式模板

```
{核心关键词} - {补充描述} | {品牌名}
```

### 规则

1. **长度**: 50-60 字符（英文），25-30 字符（中文）
2. **关键词位置**: 核心关键词放在最前面
3. **品牌名**: 始终放在最后，用 `|` 分隔
4. **避免**: 关键词堆砌、重复词汇

### 示例

✅ 正确:
```
Nano Banana Pro API - AI Image Generation | Ruxa.ai
```

❌ 错误:
```
Nano Banana Pro - Gemini 3 Pro AI Image Generation API | Ruxa.ai
```
原因: 太长，关键词堆砌

## Description 规范

### 格式模板

```
{核心功能描述}。{技术特点/优势}。{行动号召}。
```

### 规则

1. **长度**: 150-160 字符（英文），75-80 字符（中文）
2. **关键词**: 自然包含核心关键词 1-2 次
3. **结构**: 功能 + 特点 + CTA
4. **避免**: 过度营销语言、关键词堆砌

### 示例

✅ 正确:
```
Professional AI image generation API. 4K output, text rendering, multi-image support. Start free with Ruxa API.
```

## 模型页面模板

### Title
```
{Model Name} API - {核心功能} | Ruxa.ai
```

### Description
```
{Model Name} API for {use case}. {Key features}. {CTA}.
```

### 示例

**Nano Banana Pro**
- Title: `Nano Banana Pro API - AI Image Generation | Ruxa.ai`
- Description: `Nano Banana Pro API for professional image generation. 4K output, text rendering, multi-image support. Start free.`

**Sora 2**
- Title: `Sora 2 API - AI Video Generation | Ruxa.ai`
- Description: `Sora 2 API for text-to-video generation. High-quality output, flexible duration. Get started today.`
